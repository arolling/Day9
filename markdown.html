Markdown

            
Definition

            Markdown is a lightweight markup language with plain text formatting syntax designed so that it can be converted to HTML and many other formats using a tool by the same name. Below is an example of some code that displays how Markdown is used. In order to create a header, you use the # symbol to denote size. # represents a font size of H1 and ###### represents H6 as seen below. You can choose any number of hashes in between 1 and 6.
